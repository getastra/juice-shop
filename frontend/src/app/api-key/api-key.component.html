API Key Examples

Introduction

This page provides real-world examples of credentials and api keys that are prone to being accidentally leaked by developers. The algorithm detects private keys, OAuth tokens and numerous other credentials that use deterministic string formats.

Twilio

Twilio is another API that's in wide use. It offers the use of API Key and SID key to access it's services.

Twilio API Key

TWILIO_API_KEY = SK2a0747eba6abf96b7e3c3ff0b4530f6e

Twilio SID Key

TWILIO_SID_KEY = ACdbc12abd239bde2cfeDA32CdAe2DecBD

Mailgun

Mailgun API allows you to send emails and perform other actions linked to your Mailgun account programmatically. The API key has full control over your account.

Mailgun API Key

MAILGUN_API_KEY = "key-eg8f02fb71f9ee886e96588013241ca0"

SendGrid API Key

SendGrid is a communication platform for transactional and marketing emails. It offers a REST API to programmatically send emails and perform all sort of actions with a SendGrid account.
